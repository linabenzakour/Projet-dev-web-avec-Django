À propos du projet

Projet réalisé à l'école Centrale Méditerranée

Ce site a été conçu dans le cadre d'un projet académique à l'école d'ingénieurs Centrale Méditerranée, plus précisément dans le cadre du cours Informatique : Data Driven. L'objectif principal de ce projet était de développer une application web à l'aide de Django.

Le projet se concentre sur l'utilisation des technologies web pour la gestion des données, en mettant l'accent sur l'analyse et la manipulation des données dans un environnement de développement orienté vers le Data Driven.

Les principales compétences abordées pendant ce projet incluent l'utilisation des frameworks Django pour le backend, ainsi que l'implémentation de bases de données pour gérer les informations des utilisateurs et des entités du projet.
